# Speed
roblox speed modules

## Loading speed modules
```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/Skibidi50-lol/Speed/refs/heads/main/speed.lua"))()
```



## Settings
```lua
getgenv().speed = {
    enabled = false,     
    speed = 16,        
    control = false,
    friction = 2.0,    
    keybind = Enum.KeyCode.KeypadDivide 
}
```
